# Run in development

1. Clone the repository
2. Install dependencies
   ```
   yarn install
   ```
3. Install nest cli globally
   ```
    npm i -g @nestjs/cli
   ```
4. Deploy the database
   ```
   docker compose up -d
   ```

## Used stack

- NestJS
- MongoDB
- Docker
